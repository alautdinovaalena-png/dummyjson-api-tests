# Тестирование API DummyJSON

Автотесты для публичного API [DummyJSON](https://dummyjson.com) в рамках тестового задания.

---

## Опыт кандидата

- **Ручное тестирование:** Да
- **Нагрузочное тестирование:** Да (базовый опыт)
- **Тестирование безопасности:** Да (поверхностный опыт)

---

## Используемые инструменты

- **Postman** - создание коллекции запросов
- **Newman** - запуск тестов из командной строки
- **Node.js / npm** - среда выполнения
- **Git** - контроль версий

---

## Команда для запуска тестов

### Требования
- Установленный Node.js
- Установленный Newman: `npm install -g newman`

### Запуск
```bash
newman run DummyJSON_Tests.postman_collection.json -e DummyJSON_Environment.postman_environment.json
